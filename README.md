# githubDemo
git init없이 git clone으로 모든 git내용을 가져올 수 있음\n
git diff 명령어로 수정된 것 확인가능\n
git add를 사용하면 수정된 파일을 on stage 시킴\n
git pull을 하면 동기화가능 사용하는게 좋음 개인 레포쓸땐 pull사용할 필요 없지만 협업 시 필수 pull로 저장사항 로컬로 가져와야함\n
다른 branch에서 수정된 것을 master branch에 적용시키려면 pull request해서 merge해야함
Fork는 다른 계정의 remote repository를 자신의 github 계정에 복사해 오는 것
Clone은 remote repository를 자신의 local repository에 복사 해 오는 것
